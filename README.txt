# Pages complètes (théorie + exercice) pour Rise

Ce dossier contient 9 fichiers HTML complets : chaque page inclut le **contenu théorique** suivi de l’**exercice interactif** correspondant.
Ajoutez-les dans Rise via le bloc **Intégrer (Embed)**.

## Fichiers
00-introduction.html — Introduction + objectifs
01-section-qualite-securite.html — Section 1 (théorie + exercice matching)
02-section-fondements-ihi.html — Section 2 (théorie + exercice IHI)
03-section-smart.html — Section 3 (théorie + exercice SMART)
04-section-pareto.html — Section 4 (théorie + exercice Pareto)
05-section-driver.html — Section 5 (théorie + exercice Driver Diagram)
06-section-pdsa.html — Section 6 (théorie + exercice PDSA)
07-section-mesurer.html — Section 7 (théorie + exercice Run chart)
08-conclusion-quiz.html — Conclusion + quiz final

## Intégration
1) **Hébergez** ces fichiers (https).
2) Dans Rise, ajoutez **Embed (Code d'intégration)** et collez par ex. :
<iframe src="https://votre-site/01-section-qualite-securite.html" width="100%" height="1200" style="border:0;border-radius:12px" allow="clipboard-write"></iframe>
   • Ajustez la hauteur selon la page (1000–1600 px).
3) Pour contrôler la progression, placez après l'iframe un **Knowledge Check** et un **bloc Continuer** configuré sur « Tous les blocs ci-dessus ».
